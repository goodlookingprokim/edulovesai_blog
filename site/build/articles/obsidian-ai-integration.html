Obsidian AI 통합 종합 인덱스

📋 개요 {#개요}

Obsidian과 생성형 AI를 결합한 고급 노트테이킹 시스템 구축을 위한 종합 가이드 모음입니다. Claude Code, MCP, 그리고 다양한 자동화 워크플로우를 통해 노트 생산성을 극대화하는 방법을 다룹니다.

🗂️ 폴더 구조 {#폴더-구조}

📁 Claude Code 연동 {#claude-code-연동}

Claude Code와 Obsidian을 연동하여 AI 기반 노트 관리 시스템 구축

핵심 가이드

  • Obsidian_Claude_Code_협업_설정_가이드 - 초기 설정부터 일상 워크플로우까지
  • Claude Code 완벽 가이드 - 실습형 노트 - Claude Code 전체 기능 마스터

📁 MCP 활용 {#mcp-활용}

Model Context Protocol을 활용한 외부 도구 통합

주요 서버 가이드

  • Claude Code CLI YouTube MCP 설치 가이드 - YouTube 콘텐츠 분석 자동화
  • YouTube MCP Server 활용 가이드 - 동영상 트랜스크립트 노트 변환
  • YouTube MCP 실전 활용 예시 - 실제 사용 사례
  • Notion_MCP_Server_활용_사례집 - Notion-Obsidian 동기화
  • n8n MCP로 워크플로우 자동 생성하기 - 워크플로우 자동화
  • Code_Context_MCP_완벽_활용_가이드 - 코드베이스 분석 통합
  • MacBook_mcp_chrome_설치_가이드 - 브라우저 자동화

📁 워크플로우 자동화 {#워크플로우-자동화}

고급 워크플로우

💡 핵심 활용 팁 {#핵심-활용-팁}

즉시 적용 가능한 Best Practices {#즉시-적용-가능한-best-practices}

  1. CLAUDE.md 규칙 파일: 모든 프로젝트에서 /init 명령으로 시작
  2. Speech-to-Text: 입력 속도 4배 향상 (Whisper Flow 추천)
  3. 태그 시스템: TAGS.md 파일로 일관된 태그 체계 유지
  4. 서브 에이전트: 병렬 처리로 연구 작업 가속화
  5. GitHub Actions: 모바일에서도 노트 자동 생성

생산성 극대화 조합 {#생산성-극대화-조합}

  • 연구 작업: Claude Code + Context7 MCP + 서브 에이전트
  • 콘텐츠 정리: YouTube MCP + 자동 태깅 + Wiki 링크
  • 프로젝트 관리: GitHub 연동 + Plan Mode + 커스텀 명령어
  • 지식 동기화: Notion MCP + 자동화 워크플로우

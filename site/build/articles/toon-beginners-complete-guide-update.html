🚀 빠른 시작 (5분) {#빠른-시작-5분}

1단계: TOON이 뭐예요? {#1단계-toon이-뭐예요}

TOON은 AI에게 데이터를 더 효율적으로 전달하는 새로운 포맷입니다.

JSON (기존 방식):

JSON 
{
  "friends": ["철수", "영희", "민준"]
}
3 lines

TOON (새로운 방식):

Code 
friends[3]: 철수,영희,민준
1 line

효과: 같은 정보인데 30~60% 더 적은 토큰 사용!

2단계: 3가지만 기억하면 돼요 {#2단계-3가지만-기억하면-돼요}

Code 
# 1. 객체 (한 사람의 정보)
이름: 김철수
나이: 30


2. 배열 (여러 항목)


친구[3]: 철수,영희,민준


3. 표 (여러 사람의 정보)


학생[2]{이름,나이}:
  김철수,30
  이영희,25
9 lines

3단계: 왜 배워야 할까요? {#3단계-왜-배워야-할까요}

이유 설명
💰 비용 절감 AI API 사용 비용 30~60% 감소
⚡ 속도 AI의 응답 시간 단축
🎯 정확도 AI가 더 정확하게 이해
📊 효율성 대용량 데이터 처리 최적화

🎯 학습 목표 체크리스트 {#학습-목표-체크리스트}

파트 1을 마친 후 {#파트-1을-마친-후}

  • TOON이 뭔지 1문장으로 설명할 수 있다
  • 왜 JSON보다 TOON이 효율적인지 알 수 있다
  • 객체, 배열, 표의 차이를 설명할 수 있다
  • 간단한 데이터를 TOON으로 만들 수 있다
  • 용어 사전의 주요 용어 10개를 이해한다
  • TOON을 사용하지 말아야 할 경우를 알 수 있다

파트 2를 마친 후 {#파트-2를-마친-후}

  • 객체를 자유롭게 다룰 수 있다
  • 배열과 표를 완벽히 이해한다
  • 따옴표 규칙을 모두 알 수 있다
  • 빈 컨테이너를 다룰 수 있다
  • 타입 변환을 이해할 수 있다
  • 실무 예제 4가지를 직접 만들 수 있다
  • CLI 도구로 자동 변환할 수 있다

파트 3을 마친 후 {#파트-3을-마친-후}

  • 구분자의 3가지 종류를 사용할 수 있다
  • Key Folding으로 토큰을 절감할 수 있다
  • 복합 구조를 다룰 수 있다
  • 레이턴시 vs 토큰 트레이드오프를 이해한다
  • 성능 최적화 전략을 적용할 수 있다
  • Playgrounds로 실시간 테스트할 수 있다

🤔 자주 묻는 질문 {#자주-묻는-질문}

Q1: 모든 데이터를 TOON으로 해야 하나요? {#q1-모든-데이터를-toon으로-해야-하나요}

A: 아니요. 균일한 테이블 데이터에 TOON이 최고입니다. 깊게 중첩된 데이터는 JSON이 더 나을 수 있어요.

Q2: Part 1만 읽어도 되나요? {#q2-part-1만-읽어도-되나요}

A: 기본을 이해하려면 가능하지만, Part 2를 해야 실제로 사용할 수 있습니다!

Q3: 얼마나 자주 사용하게 되나요? {#q3-얼마나-자주-사용하게-되나요}

A: AI를 자주 사용한다면 거의 매일! 대용량 데이터를 처리할 때마다 유용합니다.

Q4: CLI 도구를 반드시 배워야 하나요? {#q4-cli-도구를-반드시-배워야-하나요}

A: 기본 개념을 알면 손으로 작성할 수 있지만, CLI가 있으면 훨씬 편합니다.

Q5: JSON을 TOON으로 자동 변환되나요? {#q5-json을-toon으로-자동-변환되나요}

A: 네! CLI 도구로 자동 변환됩니다.

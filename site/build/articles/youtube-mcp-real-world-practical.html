Code 
graph LR
    A[YouTube URL] --> B[MCP로 데이터 추출]

    C --> D[노트 생성]
    D --> E[태그 및 링크 추가]

Plain Text

모든 코드는 실행 가능한 형태로 추출하고,
'R - Resources/프로그래밍/Python' 폴더에 저장해줘"

고급 팁 {#고급-팁}

1. 자막 언어 처리 {#1-자막-언어-처리}

  • 한국어 자막 우선 사용
  • 없을 경우 영어 자막 + 핵심 용어 번역
  • 자동 생성 자막도 활용 가능

2. 긴 영상 처리 전략 {#2-긴-영상-처리-전략}

  • 2시간 이상: 30분 단위로 섹션 분할
  • 챕터 마커 활용하여 구조화
  • 핵심 요약 먼저, 상세 내용은 접기 사용

Plain Text 
"이 영상은 자막이 없네요. 대신:
1. 영상 설명(description) 분석
2. 댓글에서 주요 내용 추출
3. 썸네일과 제목으로 주제 파악
으로 노트를 만들어줘"

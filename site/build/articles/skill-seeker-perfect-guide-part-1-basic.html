Skill Seeker가 뭔가요? {#skill-seeker가-뭔가요}

🎁 마법 같은 도구의 정체 {#마법-같은-도구의-정체}

Skill Seeker는 **"문서 사이트를 Claude AI가 이해할 수 있는 지식으로 변환해주는 자동화 도구"**입니다.

🌟 마법의 동화로 이해하기

옛날에 거대한 도서관이 있었어요. (= 문서 사이트)

  • 책이 40,000권이나 있었죠
  • 책은 모두 영어로 되어있었어요
  • 찾고 싶은 내용이 어디 있는지 몰라요

그런데 마법사 Skill Seeker가 나타나서:

  1. 📚 모든 책을 읽고
  2. 🏷️ 주제별로 정리하고
  3. 📝 핵심 내용과 예제를 뽑아내고
  4. 🎁 Claude AI가 이해할 수 있는 "선물 상자"로 포장했어요!

📋 Step 1: Python 확인 {#step-1-python-확인}

Python이 뭔가요?

프로그래밍 언어의 한 종류예요. Skill Seeker는 Python으로 만들어졌어요. 마치 한국 요리를 하려면 젓가락과 숟가락이 필요한 것처럼, Python 프로그램을 실행하려면 Python이 설치되어 있어야 해요!

확인 방법:

Bash 
# 터미널을 열고 입력:
python3 --version


결과 예시:


Python 3.11.5  ✅ 좋아요!
Python 3.9.0   ❌ 너무 낮아요 (3.10 이상 필요)
command not found  ❌ 설치 안 됨
6 lines

설치 필요하면:

  1. https://www.python.org/downloads/ 방문
  2. 큰 노란색 "Download Python" 버튼 클릭
  3. 다운로드된 파일 실행
  4. 중요! "Add Python to PATH" 체크박스 선택
  5. "Install Now" 클릭

🌿 중급자 레벨: 가상환경 만들기

가상환경(Virtual Environment)이 뭔가요?

각 프로젝트만을 위한 "독립된 작업 공간"이에요. 비유: 각자의 방에서 공부하는 것

  • 내 방(가상환경A)에서는 Python 라이브러리 A 사용
  • 동생 방(가상환경B)에서는 Python 라이브러리 B 사용
  • 서로 충돌 안 나요!
Bash 
# 1. 가상환경 생성
python3 -m venv venv


이게 뭔가요?


python3 = Python 3 사용해줘


-m venv = venv 모듈 실행해줘


venv (마지막) = "venv"라는 이름으로 폴더 만들어줘


2. 가상환경 활성화


Mac/Linux:


source venv/bin/activate


Windows:


venv\Scripts\activate


성공하면 이렇게 보여요:


(venv) YourName@Computer Skill_Seekers %


👆 앞에 (venv)가 보이면 성공!
14 lines

🌳 고급자 레벨: 의존성 설치

의존성(Dependencies)이 뭔가요?

프로그램이 작동하려면 필요한 "다른 프로그램들"이에요. 비유: 케이크를 만들려면

  • 밀가루 필요 (= requests 라이브러리)
  • 계란 필요 (= beautifulsoup4 라이브러리)
  • 설탕 필요 (= pytest 라이브러리)
Bash 
# 필수 라이브러리 설치
pip install requests beautifulsoup4 pytest

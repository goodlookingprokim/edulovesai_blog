🎭 민수의 이야기 {#민수의-이야기}

민수는 수제 키링을 파는 작은 온라인 샵을 운영하고 있습니다. 지금까지는 카카오톡으로 주문을 받고 계좌이체로 입금받았는데, 손님들이 불편해합니다.

"결제 시스템 좀 만들어주세요!"

민수는 검색해봤습니다. 웹 개발자를 고용하면 최소 수백만 원, 시간도 몇 주가 걸립니다. 포기하려던 순간, AI 코딩 도구를 발견합니다.

"코딩을 모르는 내가 결제 시스템을 만들 수 있을까?"

🎯 이 가이드가 보여줄 것 {#이-가이드가-보여줄-것}

이 가이드는 민수처럼 코딩 경험이 전혀 없는 분들도:

  • AI 도구(Cursor)를 사용하여
  • 실제 작동하는 결제 시스템을 만들고
  • 발생하는 모든 에러를 해결하는 방법을

처음부터 끝까지 보여드립니다.

💪 여러분이 얻게 될 능력 {#여러분이-얻게-될-능력}

이 가이드를 마치면:

  • ✅ 토스 페이먼츠 결제 시스템 구축
  • ✅ 결제 정보를 구글 시트에 자동 저장
  • ✅ 개발 중 발생하는 에러 스스로 해결
  • ✅ AI에게 효과적으로 질문하는 방법

걱정하지 마세요! 에러는 당연히 발생합니다. 이 가이드는 에러 해결 과정까지 모두 담고 있습니다.

바이브 코딩이란 무엇인가? {#바이브-코딩이란-무엇인가}

🎨 바이브 코딩의 정의 {#바이브-코딩의-정의}

**바이브 코딩(Vibe Coding)**은:

  • **"분위기로 코딩한다"**는 의미
  • 정확한 문법을 모르고
  • 깊은 원리를 몰라도
  • AI에게 원하는 것을 설명하면
  • AI가 알아서 코드를 작성해주는 개발 방식

🎭 식당으로 이해하기 {#식당으로-이해하기}

클라이언트 (Client) = 손님

  • 역할: 메뉴판을 보고 주문하기
  • 위치: 당신의 컴퓨터 (웹 브라우저)
  • 특징: 예쁜 화면으로 정보를 보여줌
Plain Text

│                     │

│                     │
│  [결제하기 버튼]     │

서버 (Server) = 주방

  • 역할: 주문을 받아서 처리하기
  • 위치: 멀리 떨어진 회사의 컴퓨터
  • 특징: 실제 데이터를 저장하고 처리함

3. 결제 완료 확인
4. 구글 시트에 주문 정보 저장
5. 결과를 클라이언트에 전달

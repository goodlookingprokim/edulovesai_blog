바이브코딩 라이브 - 커서맛피아가 대신 만들어드림 #3

개요

바이브코딩의 라이브 코딩 세션으로, AI 도구(Cursor)를 활용하여 실시간으로 SaaS 서비스를 개발하는 과정을 보여주는 스트리밍입니다.

주요 내용

개발 목표

  • 알바생 스케줄 관리 SaaS 개발
    • 매장 알바생들의 시프트 근무를 자동으로 조정하는 서비스
    • 팀 내 채팅 기능 포함
    • 구독 결제 시스템 구현 예정

핵심 기능

  1. 시프트 스케줄 자동 조정

    • 알바생들의 근무 시간표를 효율적으로 관리
    • 비는 시간대에 가능한 인력 중에서 선택하여 배치

  2. 요청 및 승인 시스템

    • 알바생이 시프트 변경 요청
    • 매니저가 승인/거절 결정
    • 대상 알바생도 승인 필요

  3. 팀 내 채팅 기능

    • 매장 내 직원들 간의 실시간 소통

개발 방식의 특징

  • 라이브 코딩: 실시간으로 시청자들과 함께 개발 진행
  • AI 도구 활용: Cursor(AI 코딩 어시스턴트)를 활용한 빠른 개발
  • 실용적 접근: 실제로 사용 가능한 수준의 기능 완성을 목표

부스터 신규 기능 소개

라이브 중 부스터(Booster) 서비스의 신규 기능도 소개:

  • 번거로운 문서 작업을 한 번에 처리하는 기능
  • 아직 정식 릴리즈 전 테스트 단계

시사점

  1. AI 코딩 도구의 실무 활용

    • Cursor와 같은 AI 도구로 실제 서비스 개발이 가능함을 보여줌
    • 개발 속도와 효율성이 크게 향상됨

  2. 실시간 문제 해결

    • 라이브 환경에서 즉각적인 문제 해결 과정 공유
    • 실무에서 마주치는 다양한 상황에 대한 대처 방법 학습

  3. SaaS 개발의 핵심 요소

    • 사용자 니즈 파악 (알바생 스케줄 관리)
    • 핵심 기능 우선 구현
    • 점진적 기능 확장 (결제 시스템 등)

관련 자료

  • Cursor 활용 가이드
  • AI 코딩 도구 비교 분석
  • SaaS 개발 프로세스

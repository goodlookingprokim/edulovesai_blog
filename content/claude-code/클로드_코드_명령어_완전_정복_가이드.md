## 개요
- **핵심 주제**: Claude Code의 모든 명령어와 기능을 체계적으로 정리한 완벽 가이드
- **목적**: 개발 속도 10배 향상과 효율적인 AI 코딩을 위한 실무 지침서
- **범위**: 기본 명령어부터 고급 확장 기능까지 Claude Code의 모든 기능

### Claude Code란?
터미널 화면에서 직접 명령을 내리는 방식의 AI 코딩 도구입니다. 편집기나 특정 환경에 얽매이지 않고 터미널에서 AI를 바로 활용할 수 있다는 것이 가장 큰 장점입니다.

## 주요 내용

### 기본 명령어

#### 작업 디렉토리 이동
```bash
cd [폴더 경로]
```
- 작업할 프로젝트 폴더로 이동
- 예: `cd my_project` - 해당 폴더로 이동하며, Claude의 모든 작업은 이 위치를 기준으로 진행

#### Claude 실행
```bash
# 대화형 모드 진입
claude

# 첫 질문과 함께 시작
claude [하고 싶은 말]
# 예: claude "오늘 날씨 어때?"

# 단일 응답 모드 (대화 모드 없이 답변만 출력)
claude -p [질문]
```
- `-p` 또는 `--print`: 간단한 정보만 필요할 때 유용

### 고급 제어 옵션

Claude 명령어 뒤에 `--`(더블 대시)로 시작하는 옵션으로 세밀한 제어가 가능합니다.

#### 대화 이어가기
```bash
# 직전 대화 이어가기
claude --continue
claude --c

# 대화 목록에서 선택하여 이어가기
claude --resume
```

#### 파일 및 권한 관리
```bash
# 추가 폴더 참조
claude --add-dir [폴더 경로]

# 권한 모드 설정
claude --permission-mode [모드]
# 모드: default, accept, edits-plan
# 주의: bypass-permissions는 모든 권한 요청 건너뜀
```

#### 모델 및 출력 설정
```bash
# 특정 모델 지정
claude --model opus
claude --model sonnet

# 디버깅 모드 (상세 로그)
claude --verbose

# 출력 형식 지정
claude --output-format json
```

### 대화형 모드 기본 단축키

터미널 환경의 특수한 단축키를 숙지해야 합니다:

| 단축키 | 기능 | 주의사항 |
|--------|------|----------|
| `Ctrl + C` | 작업 종료 | 윈도우의 복사와 다름 |
| `Ctrl + Enter` 또는 `\ + Enter` | 줄바꿈 | Enter는 메시지 전송 |
| `Ctrl + V` 또는 마우스 우클릭 | 붙여넣기 | 이미지/파일 경로 첨부 가능 |

### 대화형 모드 고급 기능

#### 참조 기능 (@)
```
@파일명
@폴더명/
@이미지.png
@https://example.com
```
- 프로젝트 내 파일, 폴더, 이미지, URL을 참조

#### 대화 제어
| 단축키 | 기능 |
|--------|------|
| `ESC` | 현재 답변 생성 취소 |
| `ESC` 두 번 | 대화 분기 - 이전 시점으로 돌아가기 |
| `↑/↓` | 이전 입력 명령어 불러오기 |
| `Ctrl + R` | 히스토리 검색 |
| `Shift + Tab` 또는 `Alt + M` | 파일 권한 모드 전환 |

### 내장 기능 (슬래시 명령어)

대화창에서 `/`로 시작하는 강력한 내장 기능들:

#### 도움말 및 상태
- `/help` - 모든 슬래시 명령어 목록
- `/status` - 계정 정보, 버전, AI 모델 확인
- `/doctor` - 설치 상태 자가 진단

#### 비용 및 피드백
- `/cost` - 현재 세션의 토큰 사용량과 비용
- `/bug` - 버그 리포트 또는 피드백 전송
- `/release-notes` - 최신 버전 변경사항

### 프로젝트 관리 기능

#### /md - 프로젝트 가이드 생성
```
/md
```
- `CLAUDE.md` 파일 생성
- 프로젝트 구조, 코딩 컨벤션, 기술 스택 등 문서화
- Claude가 프로젝트 맥락을 이해하는 핵심 파일

#### /memory - 프로젝트 지침 편집
```
/memory
```
1. **프로젝트별 설정**: 현재 프로젝트의 `CLAUDE.md` 편집
   - 예: "이 프로젝트는 Python 3.9를 사용해"
2. **전역 설정**: 모든 프로젝트 공통 지침
   - 예: "항상 한국어로 답변해"

💡 **빠른 저장 팁**: 대화창에서 `#`를 누르고 내용 작성 후 Enter로 즉시 저장

### 대화 관리 및 설정

#### 대화 초기화
- `/clear` - 대화 기록 완전 삭제 (컨텍스트 윈도우 관리 중요!)
- `/compact` - 대화 요약 후 상세 기록 삭제

#### 대화 저장 및 전환
- `/resume` - 다른 대화로 전환
- `/export` - 대화 내용 파일 저장 또는 클립보드 복사

#### 설정 관리
```
/config
```
- 테마 설정
- 자동 요약 기능
- 모델 선택 (default는 opus 50% 사용 후 sonnet 전환)
- 파일 권한 규칙 관리

#### 동적 폴더 추가
```
/add-dir [경로]
```
- 대화 중 새로운 프로젝트 폴더 추가

### 계정 및 커스텀 명령어

#### 계정 관리
- `/upgrade` - 유료 요금제 업그레이드
- `/login` / `/logout` - 로그인/로그아웃

#### 나만의 명령어 만들기
1. 프로젝트에 `claude/commands` 폴더 생성
2. 마크다운 파일로 명령어 작성
3. 파일명이 곧 명령어

```bash
# 예시: claude/commands/check.md
"버그 관점에서 이 코드를 리뷰해 줘"
```
사용: `/check 파일경로`

### 확장성 및 연동

#### MCP (Model Context Protocol)
Claude의 플러그인 시스템으로 무한한 확장 가능:
- 외부 프로그램/서비스 연결
- 다른 AI와 협업
- 데이터베이스 조회
- 예: `context-7` MCP로 외부 도구 추가

#### GitHub 연동
- `review-pr-comments` - PR 리뷰 자동화
- `install-github-app` - GitHub 이슈에서 `@app-claude` 태그로 자동화

#### IDE 연동
- VS Code, Cursor 등과 연동
- 터미널에서 `claude` 실행으로 편집기 UI + Claude AI 동시 활용

### 사용량 및 비용 확인

```bash
ccu-page
```
- Claude Code 사용량 대시보드
- 예상 비용 실시간 확인
- 터미널에서 언제든 조회 가능

## 실무 활용 팁

### 효율적인 워크플로우
1. **프로젝트 시작 시**: `/md`로 프로젝트 컨텍스트 설정
2. **긴 작업 시**: 주기적으로 `/compact` 또는 `/clear`로 컨텍스트 관리
3. **반복 작업**: 커스텀 명령어로 자동화
4. **협업 시**: GitHub 연동으로 팀 생산성 향상

### 주의사항
- 컨텍스트 윈도우가 가득 차면 예상치 못한 동작 발생 가능
- `bypass-permissions` 옵션은 신중히 사용
- 비용 관리를 위해 정기적으로 `/cost` 확인

### 문제 해결
- 설치 문제: `/doctor` 실행
- 버그 발생: `/bug`로 리포트
- 성능 저하: `/clear`로 대화 초기화

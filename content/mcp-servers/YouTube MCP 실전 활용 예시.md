# YouTube MCP 실전 활용 예시

## Claude Code에서 YouTube MCP 사용법

### 1. 기본 사용법
YouTube MCP Server를 사용하여 영상 정보를 가져오려면 다음과 같이 요청하세요:

```
"YouTube URL [https://youtube.com/watch?v=...] 영상의 자막과 정보를 가져와서 
Obsidian 강의 노트를 만들어줘"
```

### 2. 사용 가능한 MCP 서버
- **youtube**: YouTube 영상 정보 및 자막 추출

## 실전 명령 예시

### 단일 영상 노트 생성
```
다음 YouTube 영상의 내용을 분석해서 Obsidian 강의 노트를 만들어줘:
https://www.youtube.com/watch?v=VIDEO_ID

1. YouTube MCP를 사용해 영상 정보와 자막을 추출
2. 'YouTube 강의 노트 템플릿' 사용
3. 시간대별로 주요 내용 정리
4. 코드가 있다면 별도로 추출
```

### 프로그래밍 튜토리얼 분석
```
이 Python 튜토리얼 영상을 분석해줘:
[YouTube URL]

특히 다음 내용을 중점적으로:
- 사용된 모든 코드 예제 추출
- 주요 함수와 메서드 정리
- 실습 따라하기 체크리스트 생성
```

### 컨퍼런스 발표 정리
```
이 기술 컨퍼런스 발표를 정리해줘:
[YouTube URL]

포함할 내용:
- 발표자 소개 및 배경
- 핵심 메시지 3-5개
- 시연된 데모나 사례
- Q&A 세션 주요 질문
```

### 온라인 강의 시리즈
```
이 강의 시리즈의 첫 번째 영상을 정리해줘:
[YouTube URL]

추가로:
- 전체 시리즈 구성 파악
- 다음 강의와의 연결점 표시
- 선수 지식 요구사항 정리
```

## 워크플로우 자동화

### 1. 표준 프로세스
```mermaid
graph LR
    A[YouTube URL] --> B[MCP로 데이터 추출]
    B --> C[템플릿에 데이터 매핑]
    C --> D[노트 생성]
    D --> E[태그 및 링크 추가]
    E --> F[Obsidian Vault에 저장]
```

### 2. 효율적인 명령 구조
```
[목적] + [YouTube URL] + [특별 요구사항] + [저장 위치]

예시:
"Python 기초 강의 https://youtube.com/... 를 정리하되,
모든 코드는 실행 가능한 형태로 추출하고,
'R - Resources/프로그래밍/Python' 폴더에 저장해줘"
```

## 고급 팁

### 1. 자막 언어 처리
- 한국어 자막 우선 사용
- 없을 경우 영어 자막 + 핵심 용어 번역
- 자동 생성 자막도 활용 가능

### 2. 긴 영상 처리 전략
- 2시간 이상: 30분 단위로 섹션 분할
- 챕터 마커 활용하여 구조화
- 핵심 요약 먼저, 상세 내용은 접기 사용

### 3. 시리즈물 관리
```markdown
[[시리즈명 - 전체 목차]]
├── [[시리즈명 - 1강]]
├── [[시리즈명 - 2강]]
└── [[시리즈명 - 3강]]
```

## 문제 해결

### 자막이 없는 경우
```
"이 영상은 자막이 없네요. 대신:
1. 영상 설명(description) 분석
2. 댓글에서 주요 내용 추출
3. 썸네일과 제목으로 주제 파악
으로 노트를 만들어줘"
```

### MCP 오류 시
```
"YouTube MCP가 작동하지 않으면:
1. URL이 올바른지 확인
2. 영상이 공개 상태인지 확인
3. 수동으로 주요 정보 입력 요청"
```

# Flutter 입문자 가이드 - 빠른 네비게이션

> **"어디서 시작해야 할까? 이 페이지가 도움될 거다."**

---

## 🚀 당신의 상황에 맞는 시작점 선택

### 1️⃣ "정말 처음이에요" 👶

```
당신의 상황:
  • Flutter가 정확히 뭔지 모른다
  • 프로그래밍 경험이 거의 없다
  • 천천히 이해하고 싶다

추천 순서:
  1️⃣ [[Flutter 입문자 완전 가이드 - 마스터 가이드]]
     (5분: 전체 구조 이해)

  2️⃣ [[Flutter 입문자 완전 가이드 Part 1]]
     (2시간: 기초 개념)
     ├─ Widget이 뭔가? 섹션부터
     └─ 핵심 개념 5가지 읽기

  3️⃣ [[Flutter 입문자 완전 가이드 Part 2]]
     (30분: 난이도별 앱)
     └─ "초급자 추천 앱" 섹션만

  4️⃣ [[Flutter 입문자 완전 가이드 Part 3]]
     (2시간: 1주차만)
     └─ "1주차: 기초 다지기" 따라하기

시간 투자: 약 5시간 (1주일)
결과: 첫 To-Do 앱 완성 ✅
```

### 2️⃣ "약간의 프로그래밍 경험이 있어요" 👨‍💻

```
당신의 상황:
  • Python, JavaScript 등 기본 경험
  • 앱 개발은 처음
  • 빠르게 실전으로 나가고 싶다

추천 순서:
  1️⃣ [[Flutter 입문자 완전 가이드 Part 1]]
     (1시간: 핵심 5가지 개념만)
     ├─ "핵심 개념 5가지" 섹션
     └─ Widget, State, Hot Reload

  2️⃣ [[Flutter 입문자 완전 가이드 Part 2]]
     (1시간: 앱 카테고리 빨리 훑기)
     └─ "초급자 추천 앱" 섹션

  3️⃣ [[Flutter 입문자 완전 가이드 Part 3]]
     (2시간: 1-2주차)
     └─ To-Do → 계산기 앱 만들기

  4️⃣ [[Flutter 입문자 완전 가이드 Part 4 - 앱 사례 분석]]
     (병행: Deer 앱 분석)

시간 투자: 약 5시간 (1주일)
결과: 계산기 앱 + BLoC 패턴 이해 ✅
```

### 3️⃣ "빨리 배워서 앱을 만들고 싶어요" ⚡

```
당신의 상황:
  • 프로그래밍 경험이 충분
  • 시간이 없다
  • 실전 중심으로 배우고 싶다

추천 순서:
  1️⃣ [[Flutter 입문자 완전 가이드 Part 1]]
     (30분: 용어 사전만)
     └─ "용어 사전" A~W 빠르게 훑기

  2️⃣ [[Flutter 입문자 완전 가이드 Part 3]]
     (3시간: 1-2주차 집중)
     ├─ To-Do 앱 만들기
     └─ 계산기 앱 만들기

  3️⃣ [[Flutter 입문자 완전 가이드 Part 4 - 앱 사례 분석]]
     (1시간: Tier 1 앱들)
     ├─ Deer 분석
     ├─ GitTouch 분석
     └─ medito-app 분석

  4️⃣ 자신의 아이디어 앱 시작

시간 투자: 약 4시간 (3-4일)
결과: 작동하는 앱 2개 + 기술 이해 ✅
```

### 4️⃣ "특정 주제에 대해만 궁금해요" 🔍

```
당신의 상황:
  • 이미 기본은 알고 있다
  • 특정 기술만 배우고 싶다
  • 깊이 있게 학습하고 싶다

주제별 추천:

◆ "Widget과 UI"를 배우고 싶다
  └─ Part 1의 "개념 2: Widget" 섹션
  └─ Part 2의 "Layout" 섹션
  └─ Part 4의 "medito-app" 사례 (아름다운 UI)

◆ "상태 관리"를 배우고 싶다
  └─ Part 1의 "개념 3: State" 섹션
  └─ Part 3의 "2주차: 계산기" (setState)
  └─ Part 4의 "기술 스택별 비교" (Provider, GetX, BLoC)
  └─ Part 4의 "Deer" 앱 분석 (BLoC 패턴)

◆ "API 통신"을 배우고 싶다
  └─ Part 2의 "날씨 앱" 섹션
  └─ Part 3의 "4주차: 날씨 앱"
  └─ Part 4의 "GitTouch" 사례 (복잡한 API)

◆ "데이터 저장"을 배우고 싶다
  └─ Part 2의 "메모 앱" 섹션
  └─ Part 3의 "3주차: 메모 앱"
  └─ Part 4의 "기술 스택별 비교" (Hive, SQLite)

◆ "게임 개발"을 배우고 싶다
  └─ Part 2의 "Games & Fun" 섹션 (80개 앱!)
  └─ Part 4의 "Tier 2" (게임 앱들)

◆ "금융 앱"을 만들고 싶다
  └─ Part 4의 "Finance" 카테고리
  └─ Part 4의 "차트 라이브러리" 섹션
  └─ "Cake Wallet" 또는 "Monekin" 분석

◆ "사진/비디오"를 다루고 싶다
  └─ Part 2의 "Photo & Video" 섹션
  └─ Part 4의 "Immich" 사례 (최고 수준)

◆ "AI/ChatGPT 앱"을 만들고 싶다
  └─ Part 4의 "AI & LLMs" 카테고리
  └─ 25개 앱 중 선택
```

## 🆘 이런 상황이면?

### "처음 읽었는데 이해가 안 가요"

```
해결책:
  1️⃣ Part 1을 다시 읽기 (천천히)
  2️⃣ 용어 사전에서 어려운 단어 찾아보기
  3️⃣ "예시" 섹션을 따라 손으로 직접 코딩해보기
  4️⃣ 이해 안 되는 부분만 Part 4의 사례 앱에서 찾기
  5️⃣ 커뮤니티에 질문하기 (Flutter Korea)
```

### "코드가 너무 어려워요"

```
해결책:
  1️⃣ "간단한 코드 예시" 부터 시작하기
  2️⃣ Part 3 1주차의 기초부터 다시 따라하기
  3️⃣ 한 줄 한 줄 주석 달아가며 읽기

### "시간이 너무 부족해요"

```

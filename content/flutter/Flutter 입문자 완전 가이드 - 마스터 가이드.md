## 🎯 이 가이드의 특징

### 1. Feynman 기법 기반

```
"5살 아이도 이해할 수 있는" 수준의 설명
  ├─ 복잡한 개념 = 일상적 비유로 설명
  ├─ 실생활 예시 = 이야기로 전달
  ├─ 단계적 이해 = "왜?"를 물으면서 배우기
  └─ 재설명 = 자신의 말로 설명할 수 있을 때까지
```

### 2. 스토리텔링 방식

```
건조한 설명 ❌
  ↓
재미있는 이야기 ✅

예시:
  ❌ "Widget은 UI의 기본 단위입니다"
  ✅ "LEGO 블록처럼, 여러 Widget을 조합해서 앱을 만듭니다"
```

### 3. 상세한 코드 예시

```
각 개념마다:
  ├─ 기초 코드 (5줄)
  ├─ 실전 코드 (20줄)
  └─ 고급 코드 (50줄+)

모두 직접 실행 가능한 예시!
```

### 4. 시각적 정보

```
표, 다이어그램, 플로우차트, 목록
  ├─ 텍스트만으로는 부족함
  ├─ 한눈에 비교할 수 있게
  └─ 기억하기 쉽게
```

### 5. 용어 사전

```
어려운 단어는 전혀 사용 안 함
  ├─ 꼭 필요한 용어만 소개
  ├─ Part 1의 용어 사전 참고
  └─ 처음 나올 때 쉽게 설명
```

## 📖 각 Part 상세 설명

### Part 1: 기초 개념 (1주일)

**목표:** Flutter의 핵심 개념을 이해하기

```
학습 내용:
  □ Flutter란 정확히 무엇인가
    → 요리사 비유로 이해
    → 기업들이 왜 선택하는가

  □ 왜 배워야 할까
    → 경제적 이점 (시간 = 돈)
    → 기업들의 선택 사례

  □ 핵심 개념 5가지
    1️⃣ Dart 언어
       → 변수, 함수, 클래스
    2️⃣ Widget (위젯)
       → UI의 기본 단위
    3️⃣ State (상태)
       → 신호등으로 이해
    4️⃣ Layout (레이아웃)
       → Column, Row, Stack
    5️⃣ Hot Reload (핫 리로드)
       → 빠른 개발의 마법

  □ 개발자 여정의 지도
    → 초급 → 중급 → 고급으로의 경로

  □ 용어 사전 (A~W)
    → 어려운 용어를 쉽게 설명
```

**시간:** 약 8시간
- 영상 학습: 3시간
- 개념 이해: 3시간
- 복습: 2시간

**결과:** Flutter의 전체 그림이 보임

### Part 2: 앱 카테고리별 분석 (2-3주일)

**목표:** 앱 종류를 이해하고 난이도별로 파악하기

```
학습 내용:

1. 난이도별 앱 분류
   ⭐☆☆☆☆ → 할일, 계산기, 메모
   ⭐⭐☆☆☆ → 날씨, 뉴스, 음악
   ⭐⭐⭐☆☆ → 소셜미디어, 전자상거래
   ⭐⭐⭐⭐☆ → 실시간 채팅, AR 기능
   ⭐⭐⭐⭐⭐ → 머신러닝, 복잡한 시스템

2. 초급자 추천 3가지
   □ To-Do 앱 (상태 개념)
     → setState, List, ListView
   □ 계산기 앱 (함수, 레이아웃)
     → 버튼 배치, 연산자
   □ 메모 앱 (데이터 저장)
     → Hive, CRUD

3. 중급자 도약 3가지
   □ 날씨 앱 (API)
     → HTTP, JSON 파싱
   □ 뉴스 앱 (복잡한 UI)
     → 이미지, 무한 스크롤
   □ 음악 플레이어 (미디어)
     → 오디오, 타이머

4. 각 카테고리별 학습 경로
   → 4주 완성 계획 제시

5. 실제 사례 분석
   → GitTouch (GitHub 클라이언트)
   → ente (사진 백업 앱)
```

**시간:** 약 12시간
- 영상: 4시간
- 코드 읽기: 5시간
- 실습: 3시간

**결과:** 자신의 수준에 맞는 앱이 보임

### Part 3: 실전 학습 로드맵 (4주+)

**목표:** 실제 앱을 만들면서 배우기

```
학습 내용:

1주차: 기초 다지기
  월-화: Dart + Widget 개념학습 (6시간)
  수-목: To-Do 앱 만들기 (6시간)
  금요: 복습 및 정리 (3시간)

2주차: 계산기로 심화
  월-화: 계산기 설계 (4시간)
  수-목: 계산기 구현 (5시간)
  금요: 테스트 및 개선 (3시간)

3주차: 메모 앱으로 데이터 저장
  주당: Hive, CRUD 학습 (15시간)

4주차: 날씨 앱으로 네트워크
  주당: API, JSON, 에러 처리 (15시간)
  → 실제 API 사용 경험

+ 심화 (5주차~):
  □ 초급자 실수 5가지 배우기
  □ 전문가 팁과 트릭
  □ 커뮤니티 참여하기
  □ 코드 리뷰 받기
  □ 번아웃 방지법
```

- 4개의 완성된 앱
- 상태 관리의 깊은 이해

### Part 4: Awesome Flutter Apps 사례 분석 (전체 기간)

**목표:** 1,000개 앱에서 배우기

```
학습 내용:

1. 프로젝트 개요
   → 2,000+ 별, 40개 카테고리, 1,000개 앱

2. 40개 카테고리 심층 분석
   TOP 10:
   1. Productivity (가장 학습 가치 높음)
   2. Lifestyle (아름다운 UI)
   3. Business (실무 패턴)
   4. Developer Tools (고급)
   5. Finance (복잡한 로직)
   6-10. Photo, Games, Social, Education, AI

3. 초급자 추천 TOP 20 앱
   Tier 1: 필수 5개
   - Deer (To-Do + BLoC)
   - GitTouch (복잡한 API)
   - medito-app (오디오 처리)
   - Immich (대규모 프로젝트)
   - AppFlowy (Notion 클론)

   Tier 2: 중급 10개
   - RustDesk, GitJournal, Cake Wallet...

   Tier 3: 빠른 학습용 5개
   - FlipCard, Convapp, SnapInk...

4. 기술 스택별 비교

   - Provider (가장 인기)

   - Hive (가장 빠름)

   1. README 읽기
   2. pubspec.yaml 분석
   3. 프로젝트 구조 이해
   4. 핵심 코드 읽기

6. 디자인 패턴

7. 커뮤니티 통계
   → 성장률, 지역별 분포, 인기 앱
```

```
  Week 3 (메모): GitJournal 분석
```

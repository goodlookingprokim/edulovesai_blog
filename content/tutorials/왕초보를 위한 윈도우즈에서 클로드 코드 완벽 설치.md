## 개요

**Claude Code**는 기존 AI 코딩 도구들과 달리 진정한 **'페어 프로그래머'** 역할을 수행하는 혁신적인 도구입니다. 허락 없이 코드를 함부로 수정하거나 엉뚱한 파일을 건드리는 기존 AI 도구들의 문제점을 해결하고, 개발자가 원하는 방향으로 정확하게 작업을 수행합니다.

### 🎯 핵심 특징
- **정확한 코드 분석**: 프로젝트 전체를 이해하고 맥락에 맞는 코드 수정
- **안전한 파일 관리**: 허가 없이 파일을 수정하지 않음
- **효율적인 디버깅**: 간단한 버그도 빠르고 정확하게 해결
- **프로젝트 메모리**: CLAUDE.md 파일을 통한 프로젝트 맥락 기억

## 설치 전 준비사항

### 🖥️ 시스템 요구사항
- **Windows 사용자**: WSL (Windows Subsystem for Linux) 필수
- **최소 메모리**: 4GB RAM 이상
- **Node.js**: 18 이상 버전
- **인터넷 연결**: 설치 및 사용 시 필요

### 📋 설치 전 체크리스트
- [ ] 컴퓨터가 시스템 요구사항을 만족하는지 확인
- [ ] 관리자 권한으로 PowerShell 실행 가능 확인
- [ ] 안정적인 인터넷 연결 확인
- [ ] Claude.ai 계정 준비 (Pro/Max 요금제 권장)

## WSL 설치 과정

### 🔧 1단계: PowerShell 관리자 권한 실행
1. **Windows 검색창**에 'PowerShell' 입력
2. **반드시 '관리자 권한으로 실행'** 선택
3. 사용자 계정 컨트롤 창에서 '예' 클릭

### 🔧 2단계: WSL 설치 명령 실행
```bash
wsl --install
```

### ⚠️ 3단계: 오류 발생 시 해결방법

#### 가상화 기능 비활성화 오류
가장 흔한 오류는 **가상화 기능이 비활성화**된 경우입니다.

**해결 방법:**
1. **필수 기능 활성화**:
```bash
dism.exe /online /enable-feature /featurename:Microsoft-Windows-Subsystem-Linux /all /norestart
dism.exe /online /enable-feature /featurename:VirtualMachinePlatform /all /norestart
```

2. **컴퓨터 재부팅**

3. **BIOS 설정 변경**:
   - 컴퓨터 부팅 시 F2/F12/Delete 키로 BIOS 진입
   - 'Virtualization Technology' 또는 'VT-x' 옵션 찾기
   - 'Enabled'로 변경 후 저장

4. **WSL 재설치**:
```bash
wsl --install
```

### 🔧 4단계: 리눅스 사용자 설정
설치 완료 후 다음 정보를 설정합니다:
- **사용자 이름**: 영문으로 입력
- **암호**: 입력 시 화면에 표시되지 않음 (정상 동작)
- **암호 확인**: 동일한 암호 재입력

## Node.js 설치

### 🚨 중요 안내
**Windows에 Node.js가 이미 설치되어 있어도 WSL에는 별도로 설치해야 합니다.**
- Windows용 Node.js ≠ WSL용 Node.js
- WSL은 독립적인 리눅스 환경이므로 별도 설치 필요

### 📦 Node.js 설치 명령어
```bash
# Node.js 최신 LTS 버전 설치
curl -fsSL https://deb.nodesource.com/setup_lts.x | sudo -E bash -
sudo apt-get install -y nodejs

# 설치 확인
node --version
npm --version
```

## Claude Code 설치

### ⚠️ 중요: 보안 권장사항
**공식 문서에서는 보안상의 이유로 `sudo` 사용을 권장하지 않습니다.**

### 🔧 1단계: npm 권한 설정
시스템 폴더 대신 개인 폴더에 패키지를 설치하도록 설정:

```bash
# npm 글로벌 설치 경로를 개인 폴더로 변경
mkdir ~/.npm-global
npm config set prefix '~/.npm-global'
echo 'export PATH=~/.npm-global/bin:$PATH' >> ~/.bashrc
source ~/.bashrc
```

### 🔧 2단계: Claude Code 설치
```bash
npm install -g @anthropic-ai/claude-code
```

### ✅ 3단계: 설치 확인
```bash
claude --version
```

## 초기 설정 및 로그인

### 🎨 1단계: 첫 실행 및 테마 선택
```bash
# 프로젝트 폴더로 이동
cd /path/to/your/project

# Claude Code 실행
claude
```

첫 실행 시 **테마 선택 화면**이 나타납니다:
- **다크 모드**: 어두운 배경 선호 시
- **라이트 모드**: 밝은 배경 선호 시

### 🔐 2단계: 로그인 방법 선택
**Claude.ai Pro/Max 사용자 (권장)**:
- 추가 비용 없이 사용 가능
- 첫 번째 옵션 선택

**API 키 사용자**:
- 별도의 API 키 필요
- 사용량에 따른 요금 부과

### 🌐 3단계: 웹 인증 과정
1. **터미널에 표시된 URL을 웹 브라우저에서 열기**
2. **'Authorize' 버튼 클릭**
3. **인증 코드 복사**
4. **터미널에 인증 코드 붙여넣기**
5. **로그인 완료**

## 첫 실행 및 기본 사용법

### 📊 프로젝트 분석
Claude Code의 강력한 프로젝트 분석 기능을 체험해보세요:

```
프로젝트를 분석해줘
```

Claude Code는 다음과 같은 분석을 제공합니다:
- **프로젝트 구조 파악**
- **사용된 기술 스택 분석**
- **코드 품질 평가**
- **개선 사항 제안**

### 🧠 프로젝트 메모리 생성
```
/init
```

이 명령어는 **CLAUDE.md 파일**을 자동 생성합니다:
- 프로젝트 핵심 정보 저장
- 개발 명령어 정리
- 구조와 컨텍스트 기록
- 향후 대화에서 프로젝트 맥락 유지

### 📝 CLAUDE.md 파일 예시
```markdown
# 프로젝트 이름

## 프로젝트 개요
- 기술 스택: React, Node.js, MongoDB
- 주요 기능: 사용자 인증, 데이터 관리
- 개발 환경: WSL, VSCode

## 개발 명령어
- 개발 서버 시작: `npm run dev`
- 빌드: `npm run build`
- 테스트: `npm test`

## 프로젝트 구조
src/
├── components/
├── pages/
├── utils/
└── styles/
```

## 유용한 명령어

### 🔧 기본 명령어
```bash
# 도움말 보기
/help

# 대화 내용 지우기
/clear

# 프로젝트 초기화
/init

# Claude Code 종료
exit
# 또는
Ctrl + C
```

### 🔄 대화 연속성 명령어
```bash
# 이전 대화 이어서 하기
claude -c

# 최근 대화 복원
claude -r

# 특정 대화 복원
claude --resume [session-id]
```

## Windows 사용자 꿀팁

### 🎯 빠른 WSL 접근 방법
매번 복잡한 리눅스 경로를 찾지 말고 이렇게 하세요:

1. **Windows PowerShell에서 프로젝트 폴더로 이동**:
```bash
cd C:\Users\YourName\Documents\MyProject
```

2. **현재 위치에서 WSL 실행**:
```bash
wsl
```

3. **바로 Claude Code 실행**:
```bash
claude
```

### 📂 경로 변환 이해
- **Windows 경로**: `C:\Users\YourName\Documents\MyProject`
- **WSL 경로**: `/mnt/c/Users/YourName/Documents/MyProject`

WSL에서 Windows 파일에 접근할 때는 `/mnt/c/` 접두어를 사용합니다.

## 문제 해결

### 🔍 일반적인 문제 해결 방법

#### 1. 권한 오류
```bash
# 오류 예시
EACCES: permission denied

# 해결 방법
sudo chown -R $(whoami) ~/.npm
```

#### 2. 네트워크 오류
```bash
# DNS 문제 해결
sudo echo "nameserver 8.8.8.8" > /etc/resolv.conf
```

#### 3. Node.js 버전 문제
```bash
# Node.js 버전 확인
node --version

# 최신 버전으로 업데이트
sudo apt update && sudo apt upgrade nodejs npm
```

### 🆘 AI 도움 받기
설치 중 오류가 발생하면:

1. **에러 메시지 스크린샷 촬영**

   ```
   Claude Code 설치 중 이런 에러가 발생했습니다:
   [에러 메시지 붙여넣기]
   
   어떻게 해결해야 할까요?
   ```

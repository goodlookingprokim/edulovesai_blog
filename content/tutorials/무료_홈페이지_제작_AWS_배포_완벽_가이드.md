- [ ] 실제 URL로 접속 테스트

### 💡 프로 팁
> S3는 실제 서비스 중인 많은 웹사이트가 사용하는 검증된 배포 방식입니다. 안심하고 사용하세요.

### 🔐 AWS S3 버킷 정책 설정 (필수)
버킷 정책 설정에서 다음 JSON 코드를 사용하세요:

```json
{
    "Version": "2012-10-17",
    "Statement": [
        {
            "Sid": "PublicReadGetObject",
            "Effect": "Allow",
            "Principal": "*",
            "Action": "s3:GetObject",
            "Resource": "버킷 주소/*"
        }
    ]
}
```

**⚠️ 주의사항**: 
- `"Resource": "버킷 주소/*"` 부분에서 "버킷 주소"를 실제 버킷 ARN으로 교체
- 버킷 ARN은 버킷 속성에서 복사 가능 (예: `arn:aws:s3:::my-bucket-name/*`)

## 📝 강의에서 사용된 AI 프롬프트 모음

### 🏢 랜딩페이지 제작
```
가상의 기업 홍보자료를 만들어서 introduction.md 파일에 저장해줘.
@introduction.md 이 파일은 기업을 소개하는 문서야. 이 문서를 기반으로 기업 홍보용 홈페이지를 만들 거야.
HTML, CSS, Javascript만 사용해서 기업 홍보용 홈페이지를 개발해줘.
최근 유행하는 디자인 시스템을 적용해서 밝고 모던한 디자인으로 만들어줘.
결과물은 별도의 폴더에 저장해줘.
```

### 🏨 숙박업소 예약 페이지
```
가상의 숙박업소 예약 홈페이지를 만들어서 별도의 폴더에 저장해줘.
밝은 느낌의 세련된 디자인을 적용해줘.
```

### 💣 지뢰찾기 게임
```
HTML, CSS, Javascript만 사용해서 지뢰찾기 게임을 개발해줘.
결과물은 별도의 폴더를 생성해서 저장해줘.
```

### 🧱 벽돌깨기 게임
```
HTML, CSS, Javascript만 사용해서 벽돌깨기 게임을 만들어서 별도의 폴더에 저장해줘.
고전적인 게임이지만 디자인을 현대화 시켜서 제작해줘.
```

### 🎯 프롬프트 작성 팁

2. **디자인 방향 제시**: "밝고 모던한", "세련된", "현대화된" 등
3. **파일 구조 요청**: "별도의 폴더에 저장해줘"
4. **구체적인 기능 요청**: 게임의 경우 게임명만 명시해도 충분

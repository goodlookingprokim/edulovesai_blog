## 🚀 빠른 시작 (5분)

### 1단계: TOON이 뭐예요?

TOON은 **AI에게 데이터를 더 효율적으로 전달하는 새로운 포맷**입니다.

**JSON** (기존 방식):
```json
{
  "friends": ["철수", "영희", "민준"]
}
```

**TOON** (새로운 방식):
```toon
friends[3]: 철수,영희,민준
```

**효과**: 같은 정보인데 **30~60% 더 적은 토큰** 사용!

### 2단계: 3가지만 기억하면 돼요

```toon
# 1. 객체 (한 사람의 정보)
이름: 김철수
나이: 30

# 2. 배열 (여러 항목)
친구[3]: 철수,영희,민준

# 3. 표 (여러 사람의 정보)
학생[2]{이름,나이}:
  김철수,30
  이영희,25
```

### 3단계: 왜 배워야 할까요?

| 이유 | 설명 |
|------|------|
| 💰 **비용 절감** | AI API 사용 비용 30~60% 감소 |
| ⚡ **속도** | AI의 응답 시간 단축 |
| 🎯 **정확도** | AI가 더 정확하게 이해 |
| 📊 **효율성** | 대용량 데이터 처리 최적화 |

## 🤔 자주 묻는 질문

### Q1: 모든 데이터를 TOON으로 해야 하나요?
**A**: 아니요. 균일한 테이블 데이터에 TOON이 최고입니다. 깊게 중첩된 데이터는 JSON이 더 나을 수 있어요.

### Q2: Part 1만 읽어도 되나요?
**A**: 기본을 이해하려면 가능하지만, Part 2를 해야 실제로 사용할 수 있습니다!

### Q3: 얼마나 자주 사용하게 되나요?
**A**: AI를 자주 사용한다면 거의 매일! 대용량 데이터를 처리할 때마다 유용합니다.

### Q4: CLI 도구를 반드시 배워야 하나요?
**A**: 기본 개념을 알면 손으로 작성할 수 있지만, CLI가 있으면 훨씬 편합니다.

### Q5: JSON을 TOON으로 자동 변환되나요?
**A**: 네! CLI 도구로 자동 변환됩니다.

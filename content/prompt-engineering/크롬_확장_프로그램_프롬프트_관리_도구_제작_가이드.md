## 배경 및 필요성

### 현재의 문제점
- ChatGPT, Claude, Gemini 등 다양한 AI 서비스에서 동일한 프롬프트를 반복 사용
- 매번 복사-붙여넣기하는 비효율적인 작업 흐름
- 프롬프트 버전 관리와 동기화의 어려움
- 여러 기기 간 프롬프트 공유의 불편함

### 해결 방안
- 중앙화된 프롬프트 데이터베이스 구축
- 단축키를 통한 빠른 접근
- 태그 기반 분류 및 검색 기능
- 모든 기기에서의 실시간 동기화

## 핵심 기능 및 특징

### 주요 기능
1. **원격 프롬프트 DB 연동**: Google Sheets를 활용한 중앙 관리
2. **태그별 분류**: 효율적인 프롬프트 조직화
3. **검색/필터링**: 빠른 프롬프트 검색
4. **바로 붙여넣기**: 단축키(ALT+P) 지원
5. **다크모드 지원**: 사용자 환경에 맞춘 UI

### 동적 변수 지원
- `{{date}}`: 현재 날짜 자동 삽입
- `{{clipboard}}`: 클립보드 내용 삽입
- `{{selection}}`: 선택된 텍스트 삽입

## Google Sheets 데이터베이스 구축

### 시트 구조 설정
1. **새 스프레드시트 생성**
2. **시트 이름**: `prompts` (정확히 일치해야 함)
3. **헤더 구성** (첫 번째 행):
   ```
   id | title | tags | body | updatedAt | token
   ```

### 예시 데이터
```
uuid-001 | 요약(핵심5) | summary | 다음 텍스트를 핵심 5개로 요약:\n\n{{selection}} | 2025-01-09T12:00:00Z | mySecretToken
uuid-002 | 번역-문체보존 | translate | 다음 텍스트를 문체를 살려 번역:\n\n{{selection}} | 2025-01-09T12:00:00Z | mySecretToken
```

### 필드 설명
- **id**: 고유 식별자 (UUID 권장)
- **title**: 프롬프트 제목
- **tags**: 분류용 태그 (쉼표로 구분)
- **body**: 실제 프롬프트 내용
- **updatedAt**: 마지막 수정 시간
- **token**: 보안 토큰 (비공개 프롬프트용)

## Google Apps Script API 개발

### Apps Script 설정
1. Google Sheets 메뉴 → 확장 프로그램 → Apps Script
2. `Code.gs`에 다음 코드 작성:

```javascript
function doGet(e) {
  const tokenParam = e.parameter.token || "";
  const sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName("prompts");
  const data = sheet.getDataRange().getValues();
  const headers = data.shift();
  
  const result = data.map(row => {
    let obj = {};
    headers.forEach((h, i) => obj[h] = row[i]);
    return obj;
  }).filter(r => !r.token || r.token === tokenParam);
  
  return ContentService.createTextOutput(JSON.stringify({ prompts: result }))
    .setMimeType(ContentService.MimeType.JSON);
}
```

### 배포 설정
1. **배포** → **새 배포** → **웹 앱**
2. **설정**:
   - 실행 사용자: 나
   - 액세스 권한: 모든 사용자
3. **URL 복사** (나중에 확장 프로그램에서 사용)

### manifest.json 작성
```json
{
  "manifest_version": 3,
  "name": "Prompt Dock",
  "version": "1.0.0",
  "description": "나만의 프롬프트를 검색하고 즉시 삽입",
  "permissions": [
    "storage",
    "activeTab",
    "scripting",
    "clipboardWrite",
    "clipboardRead"
  ],
  "action": {
    "default_popup": "popup.html",
    "default_icon": {
      "16": "icons/icon16.png",
      "32": "icons/icon32.png",
      "48": "icons/icon48.png"
    }
  },
  "icons": {
    "16": "icons/icon16.png",
    "32": "icons/icon32.png",
    "48": "icons/icon48.png",
    "128": "icons/icon128.png"
  },
  "commands": {
    "_execute_action": {
      "suggested_key": {
        "default": "Alt+P"
      },
      "description": "프롬프트 도크 열기"
    }
  },
  "options_ui": {
    "page": "options.html",
    "open_in_tab": true
  }
}
```

## 아이콘 디자인 및 제작

### 디자인 원칙
1. **단순성**: 16px에서도 식별 가능
2. **여백**: 캔버스 대비 10-14% 여백
3. **색상**: 2-3색 이내 사용
4. **대비**: 다크/라이트 모드 모두 고려

### 제작 도구
- **온라인**: favicon.io (가장 간단)
- **디자인 툴**: Canva, Figma
- **전문 툴**: Illustrator, Photoshop

### 크기별 용도
- **16×16**: 브라우저 탭
- **32×32**: 툴바 (레티나)
- **48×48**: 확장 관리 페이지
- **128×128**: Chrome 웹 스토어
- **192×192, 512×512**: 브랜딩 자산

### 테스트

- 프롬프트 불러오기 테스트
- 변수 치환 기능 검증

## 사용 방법

### 기본 사용법
1. **단축키 ALT+P**: 팝업 열기
2. **검색**: 제목 또는 태그로 검색
3. **선택**: 클릭 또는 Enter로 선택
4. **삽입**: 현재 커서 위치에 자동 삽입

### 프롬프트 관리
- **추가**: 팝업 상단 "+" 버튼
- **수정**: 각 카드의 편집 버튼
- **삭제**: 각 카드의 삭제 버튼
- **복제**: 유사한 프롬프트 빠른 생성

### 고급 기능
- **태그 필터**: 태그 칩 클릭으로 필터링
- **키보드 네비게이션**: ↑↓ 키로 이동
- **빠른 복사**: Ctrl+C로 클립보드 복사

## 문제 해결 및 FAQ

### 자주 발생하는 문제

#### 1. 아이콘이 표시되지 않음
- **원인**: 파일 경로 오류
- **해결**: manifest.json의 경로와 실제 파일 위치 확인

#### 2. API 연결 실패
- **원인**: CORS 정책 또는 권한 문제
- **해결**: Apps Script 배포 설정 재확인

#### 3. 프롬프트가 비어 있음
- **원인**: 토큰 불일치
- **해결**: 옵션 페이지와 시트의 토큰 값 확인

### 성능 최적화
1. **로컬 캐싱**: 자주 사용하는 프롬프트 캐시
2. **지연 로딩**: 필요시에만 API 호출
3. **디바운싱**: 검색 입력 최적화

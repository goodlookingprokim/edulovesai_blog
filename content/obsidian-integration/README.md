# 옵시디언 고급 지식관리 시스템

이 옵시디언 볼트는 [[PARA 시스템|PARA 방법론]]을 기반으로 구성된 **지능형 지식관리 시스템**입니다. Claude AI와 MCP 프로토콜을 통한 자동화, 고도화된 템플릿 체계, 다차원 태그 시스템을 활용하여 단순한 노트 보관을 넘어서는 **세컨드 브레인**을 구현합니다.

> **⚠️ 중요**: Claude Code나 AI 도구에게 옵시디언 관련 질문을 할 때는 **반드시 `/Users/heomin/Documents/Obsidian Vault` 폴더에 접근하여** 현재 볼트 구조와 콘텐츠를 파악한 후 답변하도록 요청하세요.

## PARA 시스템 기반 Vault 아키텍처

  - `기술 연구 및 개발/`: AI 도구 평가 및 개발 환경 관리
  - `지식관리 시스템 운영/`: 볼트 시스템 지속적 관리


```
적절한 PARA 분류 → 태그 및 메타데이터 적용 → 
기존 지식 네트워크 연결 → 검색 가능한 지식베이스 통합
```

### 3. Claude AI 활용 패턴

#### 효과적인 명령 구조
```
[작업 목적] + [구체적 지시] + [맥락 정보] + [결과 형식] + [품질 기준]
```

- `[노트명]의 내용을 요약하고 핵심 키워드를 추출해줘`

## 고급 사용 가이드라인

### 일일 워크플로우
1. **아침**: 일간 노트 생성 및 당일 계획 수립
2. **수집**: 새로운 정보는 Inbox에 임시 저장
3. **처리**: Claude AI를 활용한 정보 분석 및 분류
4. **연결**: 기존 지식 네트워크와의 링크 형성
5. **저녁**: 일일 회고 및 지식 정리

### 주간/월간 유지관리
- **주간**: Inbox 정리, 진행 중 프로젝트 상태 업데이트
- **월간**: 태그 체계 검토, 완료 프로젝트 아카이빙
- **분기**: Vault 구조 최적화, 사용하지 않는 노트 정리

### 품질 관리 원칙
1. **일관성**: 모든 노트는 표준 구조 및 메타데이터 준수
2. **연결성**: 고립된 노트(orphan notes) 최소화
3. **검색성**: 적절한 태그 및 키워드 포함
4. **실용성**: 실행 가능한 정보 위주 정리
5. **진화성**: 지속적인 개선 및 최적화

## 💡 핵심 활용 팁

### 즉시 적용 가능한 Best Practices
1. **CLAUDE.md 규칙 파일**: 모든 프로젝트에서 `/init` 명령으로 시작
2. **Speech-to-Text**: 입력 속도 4배 향상 (Whisper Flow 추천)
3. **태그 시스템**: `TAGS.md` 파일로 일관된 태그 체계 유지
4. **서브 에이전트**: 병렬 처리로 연구 작업 가속화
5. **GitHub Actions**: 모바일에서도 노트 자동 생성

### 생산성 극대화 조합
- **연구 작업**: Claude Code + Context7 MCP + 서브 에이전트
- **콘텐츠 정리**: YouTube MCP + 자동 태깅 + Wiki 링크
- **프로젝트 관리**: GitHub 연동 + Plan Mode + 커스텀 명령어
- **지식 동기화**: Notion MCP + 자동화 워크플로우
